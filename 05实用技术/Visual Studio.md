# 默认快捷键

[VS Shortcuts | Shortcut keys for Visual Studio](https://www.dofactory.com/reference/visual-studio-shortcuts)

VS 的快捷键真是好独特。。

## 编辑

-   代码提示：`Ctrl+J`
-   注释：        `CTRL+E,C` 或 `CTRL+K,C`
-   取消注释： `CTRL+E,U` 或 `CTRL+K,U`
-   格式文档： `CTRL+E,D` 或 `CTRL+K,D`
-   格式选中： `CTRL+E,F` 或 `CTRL+K,F`
