1.  `flex-basis` 在 `flex-direction` 为 row 时控制 width，为 column 时控制 height。
2.  `flex-basis` 只对 flex items 有用，并且在 flex items 使用绝对定位时无效。

注：`flex-basis` 在一些浏览器中有 bug，最好同时设置 `flex-basis` 和 `width`。
