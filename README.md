# 文章命名规范

格式：`名称 [- 标题][-N 子标题][^_^ 相关技术 1，相关技术 2，...]`

说明：

-   名称：文章所属的最主要的范畴，eg：PHP、Node、Sublime、前端自动化、
-   \- 标题：文章阐述的主要内容，eg：- 文件和文件夹处理、- 会话技术
-   \-N 子标题：文章阐述的主要内容，eg：-1 开发环境搭建、-2 编写主页
-   ^\_^ 相关技术 1，相关技术 2，...：列出文章涉及的其他关键技术，eg：^\_^ 前端，PHP、^\_^Access，VBS
